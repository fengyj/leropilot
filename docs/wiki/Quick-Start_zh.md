# 快速入门

本指南将帮助您在 5 分钟内开始使用 LeRoPilot。

## 前提条件

- 已安装 LeRoPilot（[安装指南](Installation-Guide_zh.md)）
- 互联网连接（用于下载 LeRobot 包）

## 步骤 1：启动 LeRoPilot

**桌面模式**：从应用程序菜单或桌面快捷方式启动应用程序。

**浏览器模式**：

```bash
python -m leropilot.main
```

然后在浏览器中打开 `http://localhost:8000`。

## 步骤 2：创建您的第一个环境

1. 点击侧边栏的 **"环境"**
2. 点击 **"创建新环境"**
3. 按照向导操作：
   - **名称**：输入名称（例如 "my-first-env"）
   - **LeRobot 版本**：选择版本（推荐最新版）
   - **PyTorch 版本**：选择兼容版本
   - **Python 版本**：3.10 或 3.11
4. 点击 **"创建"**

环境创建过程将：

- 创建 Python 虚拟环境
- 安装 LeRobot 和依赖项
- 配置环境

根据您的网速，这可能需要 5-10 分钟。

## 步骤 3：激活环境

创建完成后，您可以：

- **查看详情**：点击环境卡片
- **激活**：使用 UI 中显示的激活命令
- **打开终端**：启动已激活环境的终端

## 步骤 4：配置设备（可选）

1. 转到侧边栏的 **"设备"**
2. 点击 **"添加设备"**
3. 选择设备类型：
   - **机器人**：配置您的机械臂
   - **摄像头**：添加用于录制的摄像头
4. 按照设备特定的设置向导操作

## 步骤 5：开始录制（可选）

1. 导航到 **"录制"**
2. 选择您的环境和设备
3. 点击 **"开始录制"**
4. 执行您的演示
5. 完成后点击 **"停止"**

您的数据集将保存在配置的数据目录中。
